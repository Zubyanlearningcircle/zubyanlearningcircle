# Zubyan Learning Circle

A comprehensive portal for Current Affairs and AMU Entrance Preparation.

## Features

- 📰 **Daily Current Affairs Updates** - National, International, Sports, and Government Schemes
- 📚 **AMU Previous Year Papers** - Access to past entrance exam papers
- 💳 **Secure Payment Integration** - UPI payment gateway for unlocking premium content
- 🔍 **Search Functionality** - Easily find papers by year
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile devices

## Target Audience

- UPSC Aspirants
- SSC Candidates
- AMU Entrance Exam Students
- CUET Applicants

## Getting Started

1. Open `index.html` in your web browser
2. Browse current affairs updates
3. Pay ₹10 to unlock AMU Previous Year Papers
4. Search and download papers by year

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage for payment status
